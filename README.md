# AI Engineering Professional Course

## Course Overview

A comprehensive, production-focused curriculum designed to transform beginners into capable AI engineers. This course emphasizes hands-on implementation, architectural design, and industry-standard practices for building real-world AI systems.

---

## Target Audience

- Software developers transitioning to AI engineering
- Computer science students seeking practical AI skills
- Technical professionals building AI-powered products
- Anyone committed to understanding modern AI system architecture

---

## Course Philosophy

This course teaches AI Engineering, not AI Research:

- Use pretrained models effectively
- Build production systems with AI components
- Focus on architecture, not mathematical theory
- Emphasize real-world problem solving

---

## Course Structure

### Module 1: Foundations of AI Engineering

**Core Topics:**
1. Your Next Big Step in AI Engineering
2. What Are Embeddings?
3. How Vector Similarity Works
4. Setting Up Environment Variables
5. Creating Your First Embedding

**Learning Outcome:** Students understand embeddings conceptually and can generate them programmatically.

**Key Skills:**
- Convert text to numerical representations
- Calculate cosine similarity
- Use embedding models in Python
- Manage environment variables securely

---

### Module 2: Vector Databases in Practice

**Core Topics:**
1. Introduction to Vector Databases
2. Supabase + pgvector Deep Dive
3. Setting Up Your Vector Database
4. Storing Embeddings in Supabase
5. Handling Dependency Issues

**Learning Outcome:** Students can design and manage vector storage infrastructure.

**Key Skills:**
- Set up and configure vector databases
- Store high-dimensional embeddings efficiently
- Query vector data at scale
- Troubleshoot common database issues

---

### Module 3: Semantic Search Systems

**Core Topics:**
1. What Is Semantic Search?
2. Similarity Search (Cosine / L2 / Inner Product)
3. Querying Embeddings
4. Managing Multiple Matches
5. Error Handling in AI Systems

**Learning Outcome:** Students can build a production-grade semantic search API.

**Key Skills:**
- Implement similarity search algorithms
- Rank and filter search results
- Handle edge cases and errors
- Build robust search endpoints

---

### Module 4: Document Intelligence

**Core Topics:**
1. Chunking Text from Documents
2. Designing Chunking Strategies
3. Pipeline: Split → Embed → Store
4. Challenge: Build Full Ingestion Pipeline

**Learning Outcome:** Students can build a document ingestion and search system.

**Key Skills:**
- Parse and chunk documents intelligently
- Design effective chunking strategies
- Build end-to-end data pipelines
- Process diverse document formats

---

### Module 5: Conversational AI Layer

**Core Topics:**
1. Create Conversational Responses
2. Retrieval-Augmented Generation (RAG)
3. AI Chatbot Proof of Concept
4. Context Window Optimization

**Learning Outcome:** Students build a RAG-powered chatbot.

**Key Skills:**
- Integrate LLMs with vector search
- Implement RAG architecture
- Optimize context management
- Create natural conversational flows

---

### Module 6: Capstone Project

**Solo Project: PopChoice**

Build a complete AI-powered application featuring:
- Semantic search functionality
- Stored embeddings infrastructure
- Interactive chat interface
- Production-ready backend architecture
- Full deployment pipeline

**Learning Outcome:** Students demonstrate mastery by building and deploying a complete AI system.
